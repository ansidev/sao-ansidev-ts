# sao-ansidev-ts

> ansidev &#39;s scaffolding tool for TypeScript project

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao ansidev-ts new-ts-project
```

### From git

```bash
sao ansidev/sao-ansidev-ts new-ts-project
```
